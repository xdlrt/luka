## 关键场景

示例：

- 权限拒绝：工具调用被用户或规则拒绝后如何继续。
- 上下文溢出：历史消息过长时如何裁剪或压缩。

## 数据流 / 控制流

用简短步骤描述一次典型执行链路。

```text
用户输入
-> 构造系统上下文和用户上下文
-> 发起模型请求
-> 解析 tool use
-> 权限和安全检查
-> 执行工具
-> 回传 tool result
-> 继续下一轮或停止
```

如果模块不是主循环，可以改成该模块自己的控制流。
