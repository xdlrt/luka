# 附录 B　与《拆解 Claude Code》章节对照

这本深度册按硬核技术主题重组，不逐章对应科普书，但每一章都能映射回《拆解 Claude Code》的相关章节。

## 深度册 -> 科普书

| 深度册章节 | 对应科普书章节 | 关系 |
| --- | --- | --- |
| 第 1 章　查询循环的真实状态机 | 第 1 章　为什么一次对话不够 | 把科普书的“问、做、再问”下钻为 `State`、`Terminal`、`Continue`、token 预算续跑 |
| 第 2 章　工具执行框架全貌 | 第 2 章　工具：让模型动手的协议 | 把“菜单/实现分离”下钻为重型 Tool 接口、ToolUseContext、schema 投影 |
| 第 3 章　上下文工程与记忆系统 | 第 3 章　上下文与记忆；第 12 章　状态、记忆与配置治理 | 把“记住与忘记”下钻为 compact、memory prefetch、发送视图与事实来源 |
| 第 4 章　Bash 安全与权限分层 | 第 4 章　权限、沙箱与安全护栏 | 把“安全靠代码”下钻为 shell parser、Zsh 展开、权限模式、Harness 边界 |
| 第 5 章　外部协议：MCP / LSP / OAuth | 第 6 章　连接外部世界 | 把“外部服务层”下钻为传输、认证、elicitation、工具归一化 |
| 第 6 章　扩展治理：Skill 与插件 | 第 5 章　Skill；第 7 章　插件与扩展治理 | 把“按需能力”和“插件信任”下钻为加载流程、出口隔离、marketplace 治理 |
| 第 7 章　多 Agent 编排 | 第 8 章　子智能体与任务编排 | 把“分身”下钻为 AgentTool、fork、allowedTools、Task 后端、写冲突 |
| 第 8 章　桥接、远程与服务端协议 | 第 14 章　IDE、远程与服务端桥接 | 把“桥接层”下钻为消息协议、JWT/scope、断线恢复、SDK adapter |
| 第 9 章　可观测、成本与遥测 | 第 13 章　可观测、评估与追踪 | 把“事件流”下钻为 schema、脱敏、成本归属、Hook、trace->eval |
